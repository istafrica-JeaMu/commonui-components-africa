# Components Quick Start

Get started with CommonUI Components Africa in minutes.

## Installation

```bash
pnpm install @ist/commonui-components-africa
```

## Usage

```vue
<template>
  <RuneButton text="Click me" variant="primary" wcag-label="Primary action button" />
</template>

<script setup lang="ts">
import { RuneButton } from '@ist/commonui-components-africa';
</script>
```
